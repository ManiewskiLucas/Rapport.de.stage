Rapport de stage

Avec l'entreprise freelance "Alizaryana", située au Creusot, pour le stage en développement web du lundi 11 septembre au vendredi 22 septembre.

Introduction

J'ai découvert "Alizaryana" dans un article du journal de Saône-et-Loire qui mentionnait ses compétences en développement web et la création de son entreprise freelance, travaillant de chez elle.

J'ai choisi cette entreprise pour deux raisons : elle est nouvelle dans ce domaine et elle est située au Creusot. Cela m'a permis de réduire la distance et le temps de mes trajets, car l'entreprise et le DAQ 2.0 m'ont autorisé à faire du télétravail pendant mon stage.

Mes objectifs concernant ce stage étaient de confirmer mon projet et de découvrir les aspects du métier, même si, dans ce cas, il s'agissait d'une entreprise freelance qui ne propose pas exactement le même environnement et les mêmes conditions de travail que certaines grandes entreprises du même domaine.

Ce stage avait également pour but de m'améliorer et d'en apprendre plus sur les langages informatiques HTML et CSS. Ce fut nécessaire, n'ayant pas vu les bases sur la disposition de certaines balises (header, nav, main, article et aside), qui sont pratiquement obligatoires.

Présentation de l’entreprise

"Alizaryana" est une entreprise freelance situé 11 rue des abeilles, au Creusot (71200). Lancé le 1er septembre 2023, cette entreprise gérer par Anaïs Dumont, à pour domaine la création en développement web.

Anaïs avait débuté par des études de commerce, puis commencé à travailler dans le domaine bancaire. Elle a ensuite travaillé dans le domaine de l'écriture. Passionnée par la science et les nouvelles technologies, elle a fini par se tourner vers les métiers du développement web.

Grâce à son diplôme en intégration et développement web et à son professionnalisme, elle propose ses services pour la création et la production de sites internet, ainsi que pour l'optimisation de la performance, du référencement et de l'accessibilité des sites internet.

    Veille sur les outils "GitHub" et "CodePen"

  • Qu’est ce que GitHub ?
  • GitHub est un site web qui permet aux développeurs de stocker, gérer et contrôler les modifications apportées à leur code.(Gestionnaire de version)
  • Quelle est la différence entre un commit, une branche et un repository (dépôt) dans Git ?
  • Une branche est une “copie” d’un projet sur lequel on opère des modifications de code. Un commit est une sauvegarde des fichiers locaux, écrits dans le dépôt local. Repository est un stockage centralisé et organisé de données.
  • Quelles sont les meilleures pratiques pour rédiger un bon message de commit ?
  • Utilisez l'impératif / Essayer de décrire les changements sans avoir à lire le code / Utiliser le corps du message pour expliquer "pourquoi", pour "quoi", "comment" et autres détails / Eviter les messages génériques ou sans aucun contexte / Limiter le nombre de colonnes / Garder la cohérence linguistique / Mettre une majuscule au début de la phrase.
  • Qu'est-ce que le fichier README.md et pourquoi est-il important dans un projet GitHub?
  • Le fichier readme d'un projet est un fichier texte contenant des données utiles concernant un programme. Il est soit installé avec le programme, soit fourni avec le programme d'installation du programme. Il contient principalement des instructions sur l'installation du programme, l'utilisation de ses fonctions basiques et son objectif. Il est important car il donne des informations sur le projet.
  • Quels avantages auras-tu à utiliser GitHub ?
  • Faciliter la résolution des erreurs qui peuvent se produire au cours du développement / Noter les changements dans chaque version pour aider les autres collaborateurs à être au courant des modifications apportées, des tâches réalisées et celles qui restent à accomplir / Enregistrer chaque version sauvegardée en instantané, ne va donc pas répertorier les modifications sous forme de liste / Référencer d'anciens instantanés si besoin, et de nouveaux instantanés seront créés chaque fois que le projet est modifié / Facilite la collaboration à une échelle mondiale sur les projets / Tout développeur dans le monde peut récupérer des projets et y contribuer si le propriétaire du projet le permet.
  • Qu’est-ce que CodePen ?
  • C’est un site internet permettant de tester ces codes HTML CSS et JavaScript. La plateforme permet de partager vos lignes de code et projets avec d’autres utilisateurs, d’accéder aux travaux d’autres développeurs.
  • Pourquoi peut-il être un outil génial ?
  • Il a un volet pour visualiser si le code fonctionne. Trouver plein d’idée de design, trouver des fonctions css.

    Les éléments utilisés sont :

  • width : Définir la largeur d'un élément.

  • margin : Définir l'espace entre les bords extérieurs d'un élément.
  • margin-top : Définir la marge supérieure d'un élément.
  • margin-bottom : Définir la marge inférieur d'un élément.

  • margin-right : Définir la marge droite d'un élément.
  • background-image : Définir une image de fond pour un élément.
  • background-color : Définir la couleur d'arrière-plan d'un élément.
  • font-family : Spécifier la police de caractères (ou la famille de polices) à utiliser pour afficher le texte à l'intérieur d'un élément.
  • font-size : Définir la taille de la police de caractères à utiliser pour afficher le texte à l'intérieur d'un élément.
  • border : Définir une bordure autour d'un élément.
  • border-bottom : Définir la bordure ou le trait en bas d'un élément.
  • color : Définir la couleur du texte à l'intérieur d'un élément
  • position : Contrôler la position d'un élément.
  • text-align : Définir l'alignement horizontal du contenu textuel à l'intérieur d'un élément.
  • object-fit : Spécifier comment une image ou un élément de remplacement doit être ajusté à l'intérieur de son conteneur, en termes de dimensionnement et de découpage.
  • filter : Permet d'appliquer des effets visuels aux éléments.
  • top : Définir la distance entre le bord supérieur d'un élément positionné et le bord supérieur de son contenant dans une mise en page web.
  • left : Définir la distance entre le bord gauche d'un élément positionné et le bord gauche de son contenant dans une mise en page web.
  • transform : Appliquer des transformations 2D et 3D à un élément, tel qu'un texte, une image, un div, ou tout autre contenu, en modifiant sa position, sa rotation, sa mise à l'échelle et d'autres propriétés visuelles.
  • text-transform : Contrôler la transformation du texte à l'intérieur d'un élément.
  • display : Contrôler comment un élément HTML est rendu ou affiché dans le modèle de boîte du navigateur.
  • flex-wrap : Définir comment les éléments enfants d'un conteneur flexible doivent être distribués ou enroulés sur plusieurs lignes ou colonnes en fonction de l'espace disponible.
  • flex-direction : Définir la direction dans laquelle les éléments d'un conteneur flexible sont disposés le long de l'axe principal du conteneur.
  • padding : Définir l'espacement entre le contenu d'un élément HTML et ses bords (marges intérieures).
  • padding-bottom : Utilisée pour définir la taille de l'espacement entre le contenu d'un élément HTML et son bord inférieur.
  • list-style-type : Spécifier le type de marqueur ou de puce qui est affiché avant chaque élément d'une liste.
  • cursor : Définir le type de curseur de la souris qui sera affiché lorsque l'utilisateur survole un élément interactif.
  • rotate : Position en degrés.
  • gap : Utilisée dans le contexte des mises en page flexibles et de grille pour spécifier l'espacement entre les éléments d'un conteneur.
  • align-items : aligner verticalement les éléments enfants à l'intérieur d'un conteneur flexible ou d'un conteneur en grille le long de l'axe transversal du conteneur.

    • Début d'un deuxième jeu pour apprendre à utiliser Flexbox en CSS

    Les éléments utilisés sont :

  • flex-direction : row: Les éléments sont disposés dans la même direction que le texte / row-reverse: Les éléments sont disposés dans la direction opposée au texte / column: Les éléments sont disposés de haut en bas / column-reverse: Les éléments sont disposés de bas en haut
  • Justify-content : flex-start: Les éléments s’alignent au côté gauche du conteneur / flex-end: Les éléments s’alignent au côté droit du conteneur / Center: Les éléments s’alignent au centre du conteneur / Space-between: Les éléments s’affichent avec un espace égal entre eux / Space-around: Les éléments s'affichent avec un espacement égal à l'entour d'eux
  • align-items : flex-start: Les éléments s'alignent au haut du conteneur / flex-end: Les éléments s'alignent au bas du conteneur / center: Les éléments s'alignent au centre vertical du conteneur / baseline: Les éléments s'alignent à la ligne de base du conteneur / stretch: Les éléments sont étirés pour s'adapter au conteneur
  • flex-grow : Spécifie la quantité d'espace restant dans un conteneur flexible qui doit être attribuée à l'élément. Nombre compris entre 0 et 1.
  • flex-shrink : Spécifie la quantité d'espace restant dans un conteneur flexible qui doit être retirée de l'élément. Nombre compris entre 0 et 1.
  • flex-basis : Spécifie la taille initiale d'un élément flexible. Peut être une longueur, un pourcentage, ou la valeur auto.
  • order : Définir un ordre. -1 / 0 / 1 etc... doit être utilisé.
  • flex-wrap : Nowrap: tous les éléments sont tenus sur une seul ligne / Wrap: les éléments se mettent sur plusieurs lignes au besoin / Wrap-reverse: les éléments se mettent sur plusieurs lignes dans l’ordre inversé
  • align-content : flex-start: les lignes sont alignées au début de l'axe transversal / flex-end: les lignes sont alignées à la fin de l'axe transversal / center: les lignes sont centrées le long de l'axe transversal / space-between: les lignes sont réparties de manière uniforme le long de l'axe transversal, avec un espace égal entre elles / space-around: les lignes sont réparties de manière uniforme le long de l'axe transversal, avec un espace égal autour d'elles / stretch (valeur par défaut): les lignes s'étirent pour occuper tout l'espace disponible le long de l'axe transversal
  • flex-shorthand : Propriété raccourcie qui permet de définir les trois propriétés flexibles flex-grow, flex-shrink et flex-basis en une seule fois. Nombres compris entre 0 et 1, ou la valeur auto.
